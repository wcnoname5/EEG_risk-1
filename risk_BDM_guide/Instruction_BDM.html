{{ block title }} 實驗說明(五)：介紹抽獎方式 {{ endblock }} {{ block content }}

你必須在下表（如下圖）中選擇其中一行，告知您的答案。

每回合的報酬將會由您的答案，和一電腦事先決定的抽獎決定。

本實驗有兩種抽獎方式，以下先介紹兩種抽獎的報酬決定方式。

隨機抽獎：

每回合電腦會事先產生一抽獎，其中獎可能性（隨機中獎可能性）介於 1/100 到 100/100 之間。

而每回合您事先不會知道此中獎可能性為何。

紅罐抽獎：

如果本回合的答案罐是紅罐，即可中獎，反之則無中獎。

兩種抽獎方式若中獎，都可以獲得200法幣。沒中獎則獲得 0 法幣。

下一頁將介紹如何決定抽獎方式。
