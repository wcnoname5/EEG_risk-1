{{ block title }} 實驗說明(三)：頻率判斷(報酬決定) {{ endblock }} {{ block content }}

本實驗有兩種報酬方式：紅罐抽獎 與隨機抽獎。

而每回合報酬方式會由您在下表的回答，與電腦產生的一亂數 N (介於0-100) 來決定。

以下先介紹兩種報酬方式如何實行，再介紹每回合如何決定使用哪種報酬方式。

紅罐抽獎：紅罐抽獎中，電腦會從上述 100 筆相同結果的資料中選中一筆，如果此資料來自紅罐，本回合您獲得 200 法幣；如果此資料來自藍罐，本回合您獲得 0 法幣。

隨機抽獎：隨機抽獎中，電腦會再產生一個亂數 B (介於0-99)，如果此數字小於N，本回合您獲得 200 法幣；如果此數字大於或等於 N，本回合您獲得 0 法幣。

因此，您的回答代表您認為紅罐抽獎獲得 200 法幣的可能性；而 N 的大小決定了隨機抽獎獲得 200 法幣的可能性。

電腦會根據您的回答，以及 N 的大小評估兩種抽獎得到 200 法幣的可能性，並且為您選擇可能性較高的抽獎方式實現。

舉例：

假設您認為本回合中，100 筆資料中有 40 次來自紅罐，也在表中選擇「40 次」，代表您認為紅罐抽獎中獎的可能性為 40/100。

如果電腦選中亂數為 60 ，本回合您隨機抽獎中獎的可能性為 60/100 ，比起您認為紅罐抽獎的可能性還要高，因此本回合將使用隨機抽獎。

而如果電腦選中亂數為 30 ，紅罐抽獎中獎的可能性比較高，因此本回合將使用紅罐抽獎。
